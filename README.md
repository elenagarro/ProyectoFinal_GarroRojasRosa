# Sistema de Matrícula Universitaria - Proyecto Final

## Universidad Fidélitas
Curso: Programación Avanzada

## Integrante

- Rosa Garro Rojas

## Funcionalidades implementadas

- Registro e inicio de sesión de usuarios con ASP.NET Identity.
- Gestión de Carreras (CRUD).
- Gestión de Cursos (CRUD).
- Gestión de Estudiantes (CRUD).
- Base de datos desarrollada con Entity Framework Core (Code First).
- Migraciones de la base de datos.
- Interfaz desarrollada con Bootstrap.

## Tecnologías utilizadas

- ASP.NET Core MVC
- Entity Framework Core
- ASP.NET Identity
- SQL Server LocalDB
- Bootstrap 5
- Git y GitHub
